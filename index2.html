Présentation du projet JAVASCRIPT

Objectif : Sujet : Un système de réservation de salles, amphi, stand d’événements dans les différentes infrastructure du campus fiable, rapide pour les élèves et les professeurs. Objectifs : Créer une carte interactive qui permet de sélectionner les différents bâtiments du campus. une fenêtre qui se trouve déjà sur les différents bâtiments pour indiquer combien de salles sont disponibles. Lorsque l’on appuie sur un bâtiment, un “pop-up” s’ouvre et nous indique des informations: salles disponibles et la durée de chaque réservation de chaque salle, l’identité de la personne qui a réservé. Nombre de places. Indiquer l’horaire d’ouverture des salles (salles de TP, NOVA, FABLAB,RITZ,GYMNASE...) Avoir les informations des clubs. Créer une base de données (Faire le schéma relationnel). Utiliser l’agenda pour connaître les disponibilités des salles. Trouver un bon CSS. Ajouter une partie pour les concours. Ajouter les partenaires, ou les magasins les plus proches, ou les mieux notés.
